# 🎵 Music Flow

![Python Version](https://img.shields.io/badge/Python-3.10%2B-blue?style=for-the-badge&logo=python)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey?style=for-the-badge&logo=windows)

![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

**Music Flow** — це сучасний, швидкий та багатофункціональний десктопний додаток для завантаження та прослуховування музики безпосередньо з YouTube та YouTube Music. Створений на Python, він має красивий графічний інтерфейс, вбудований плеєр з аудіовізуалізатором у реальному часі, розумну локальну бібліотеку та підтримку кількох мов.

---

## ✨ Головні фічі

- **🚀 Прямі завантаження (Треки та Плейлісти)**: Просто вставте посилання на відео, трек або цілий плейліст з YouTube. Програма автоматично розпізнає формат і завантажить аудіо у найвищій якості за допомогою вбудованого рушія `yt-dlp`.
- **🎧 Вбудований музичний плеєр**: Сторонні програми більше не потрібні. Слухайте завантажені треки миттєво прямо у додатку завдяки модулю `pygame`.
- **🎛️ Аудіовізуалізатор у реальному часі**: Спостерігайте, як ваша музика оживає! Неймовірно плавний візуалізатор реагує на частоти (FFT) треку, що зараз грає.
- **🖼️ Розумні метадані та обкладинки**: Програма автоматично завантажує обкладинки відео, імена виконавців, назви пісень та тексти пісень, вбудовуючи їх прямо у метадані (ID3 теги) MP3 файлу.
- **🚫 Інтеграція SponsorBlock**: Автоматично пропускає або вирізає немузичні сегменти (інтро, аутро, рекламу) із завантажених треків.
- **🎨 Красива система тем**: Повна підтримка **Темної**, **Світлої** та **Системної** тем. Інтерфейс динамічно підлаштовується під вашу ОС.
- **🌍 Багатомовність**: Інтерфейс повністю перекладено **Англійською** та **Українською** мовами, легко розширюється через JSON-файли.
- **📚 SQLite Бібліотека та Історія**: Зручне керування завантаженою музикою. Слухайте, видаляйте або переглядайте свою локальну бібліотеку в один клік завдяки швидкій базі даних SQLite.
- **⚡ Багатопотоковість**: Інтерфейс ніколи не зависає. Завантаження, обробка звуку та збір метаданих відбуваються плавно у фоновому режимі.

## 📸 Технологічний стек

- **[CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)**: Сучасний та адаптивний графічний інтерфейс.
- **[yt-dlp](https://github.com/yt-dlp/yt-dlp)**: Надійне та надшвидке ядро для завантаження аудіо з YouTube.
- **[Pygame](https://www.pygame.org/)**: Плавне відтворення звуку з мінімальною затримкою.
- **[NumPy](https://numpy.org/)**: Швидкі математичні обчислення (Швидке перетворення Фур'є) для візуалізатора звуку.
- **[Pillow](https://python-pillow.org/)**: Обробка зображень для обкладинок треків та іконок інтерфейсу.
- **[FFmpeg](https://ffmpeg.org/)**: Конвертація аудіо, вшивання метаданих та аналіз спектра.
- **SQLite**: Локальна база даних для миттєвого завантаження вашої музичної бібліотеки та історії.

## 🚀 Як встановити / запустити

### 📥 Спосіб 1: Готовий файл (.exe) - Найпростіше!
Вам не потрібно нічого встановлювати або вміти програмувати. 
1. Перейдіть на сторінку **[Releases](../../releases)** (Релізи) справа.
2. Завантажте останній файл `MusicFlow.exe`.
3. Запустіть його і насолоджуйтесь музикою!

### 🛠️ Спосіб 2: Запуск з коду (Для розробників)

Якщо ви хочете запустити програму через Python або змінити код:

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/ВАШ_ЮЗЕРНЕЙМ/MusicFlow.git
   cd MusicFlow
   ```

2. **Встановіть залежності:**
   Переконайтеся, що у вас встановлений Python 3.10+.
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустіть додаток:**
   ```bash
   python main.py
   ```

### 📦 Спосіб 3: Збірка готового файлу самостійно

Хочете поділитися додатком з друзями, у яких не встановлений Python? Ви можете зібрати повністю портативний Windows `.exe` файл!

Просто клікніть двічі по цьому бат-файлу:
```bash
build_exe.bat
```
*(Цей скрипт автоматично очистить старий кеш, встановить залежності у віртуальне середовище та збере додаток за допомогою PyInstaller).*

Після завершення збірки, готова до використання програма буде знаходитись у папці `dist/Music_Flow/`!
